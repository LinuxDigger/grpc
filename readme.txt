

本工具参考 JSON-RPC 协议，实现了使用JSON格式的远程过程调用
工作方式为：
1，编写json文件，描述远程过程调用的方法
2，使用gengrpc工具，生成代码
3，每个json文件，会生成5个文件。
  3.1 MODEL.h 负责结构体和函数声明
  3.2 cMODEL.c 客户端，结构体与JSON互转。用户不能修改
  3.3 sMODEL.c 服务端，结构体与JSON互转。用户不能修改
  3.4 ucMODEL.c 客户端，用于非阻塞方式时的结果报告。非阻塞方式时，用户需要修改
  3.5 usMODEL.c 服务端，用于服务端移植。用户需要修改
